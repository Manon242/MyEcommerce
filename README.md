# Ecommerce Product Frontend

This is a simple frontend application built using HTML, CSS, and JavaScript.
It consumes REST APIs from a Java Spring Boot backend to manage products.

## Features
- Add a new product
- Get product by ID
- Get product by name
- View all products
- Displays products in a card-based layout

## Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)

## Backend API
The frontend communicates with the backend using REST APIs.

Base URL:
https://your-backend-url.up.railway.app/products

## How to Run Locally
1. Clone the repository
2. Open `index.html` in a browser
3. Ensure backend is running on port 8080

https://your-frontend-url.vercel.app

## Notes
- Backend uses in-memory data storage (ArrayList)
- No database is used
